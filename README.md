# ⚡ Booking Hub - Full Stack Service Booking App

Welcome to **Booking Hub**, a comprehensive full-stack MERN application designed to streamline domain service expert discovery and booking processes instantly.

## 🚀 Features
* **Dynamic Category Deck:** Quick filters for Electrician, Plumber, Painter, Cleaner, Mechanic, Furniture, and AC Repair services.
* **Instant Expert Discovery:** Broadcast custom job requirements and find online professionals within your current zone.
* **Secure API Architecture:** Protected backend routes restricted from unauthorized automated clients (like Postman/Hoppscotch) using proprietary handshake verification.
* **Interactive UI:** Smooth navigation with responsive states and premium theme management.

## 💻 Tech Stack
* **Frontend:** React.js, Vite, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **State & Flow:** Axios, Context API
